# Basic Linux Commands

## pwd

Shows current directory.

Example:

```bash
pwd
```

---

## ls

Lists files and directories.

Example:

```bash
ls
```

---

## cd

Changes directory.

Example:

```bash
cd Documents
```

---

## mkdir

Creates a directory.

Example:

```bash
mkdir projects
```

---

## rm

Removes files.

Example:

```bash
rm file.txt
```
